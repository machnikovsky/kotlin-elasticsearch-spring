## How to run
```
docker-compose up
```

## Technologies used
🍅 Kotlin  
🕸️ Elasticsearch  
📊 Graphana  
🍃 Spring  
⚛ React  

## Ports
- :8080 - Kotlin Spring Boot backend port
- :3000 - React frontend port
- :9200 - Elasticsearch port
- :5601 - Kibana port
- :4000 - Graphana port
- :9090 - Prometheus port
